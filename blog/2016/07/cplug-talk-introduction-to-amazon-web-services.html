CPLUG Talk: Introduction to Amazon Web Services

17 Jul 2016 | Linux, AWS, Presentation, CPLUG

This past Tuesday, I gave an introductory presentation on Amazon Web Services at the monthly meetup for the Central Pennsylvania Linux User Group. The talk covered some of the services offered by AWS that would be of interest for those launching application in the cloud, as well as a brief discussion on security. A demo was given along with the presentation, but I was unable to record. The slides have been uploaded to SlideShare and included below.

Introduction to Amazon Web Services from James Armes

Next month I will be giving a more advanced talk where I will highlight one of the use cases we have for AWS at Acquia, and how we utilize CloudFormation for deploy it.

php-ews 0.1 Released

The first official release of php-ews has been created. This release includes support for Microsoft Exchange Server versions 2007 through 2010 SP3. Support for 2013 will be included in a future release. This release will not be continually supported, as resources will be devoted towards moving the library in a direction that makes it easier to use and implements standards such as PSR-4 and features such as composer support.
